# novel editor

`editor/` は SQLite とユーザーログインを使わない、LAN 内向けのファイルベース小説エディタです。

## 仕様

- 小説本文のルートは `novel_room/works/`
- 編集できるのは `works/` 配下の `.txt` と `.md` のみ
- `/` で `works/` の一覧を表示
- `/folder?path=...` で任意フォルダを表示
- `/edit?path=...` で本文を編集
- `/save` で自動保存
- ファイル作成、フォルダ作成、リネーム、削除に対応
- `works/` 外へのアクセスはパストラバーサル対策で拒否

## 起動方法

`editor/` へ移動して起動します。

```bash
cd editor
python3 app.py
```

Flask は `0.0.0.0:5000` で待ち受けます。

## アクセス方法

同じ PC から:

```text
http://127.0.0.1:5000/
```

同一 LAN 内の別端末から:

```text
http://<host-ip>:5000/
```

ホスト PC の IP はたとえば次で確認できます。

```bash
hostname -I
```

Linux のファイアウォールを使っている場合は、LAN 内から TCP 5000 番へ接続できる設定が別途必要です。
